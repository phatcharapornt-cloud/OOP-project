# OOP-project

วิธีการเข้าดู Project
1. ลงโปรแกรม Git ในเครื่อง
2. กด ปุ่ม Code
3. กด Copy link
4. เปิดโปรแกรม Command Prompt
5. พิมพ์ Git clone (ใส่ link ที่ copy มาก่อนหน้านี้)
6. เปิด Folder  ตรวจงานได้เลย

---------------------------------------------
อีกหนึ่งวิธี
1. กดเข้าไปที่ปุ่ม Code 
2. กด Dowload ZIP
3. นำไฟล์ ZIP ที่ทำการDowload มาแตกไฟล์
4. เปิดไฟล์แล้วสามารถ run ได้เลย
